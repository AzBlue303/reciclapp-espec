# ADR-7: Importancia de la interoperabilidad en el sistema
## Situación: 
Propuesto 
## Contexto: 
El sistema trabaja con distintos componentes que utilizan distintas tecnologías, como los sensores y servidores.
## Decisión:
Se decide mantener una interoperabilidad entre estos componentes, manteniendo una comunicacion coherente entre los componentes..
## Consecuencias:
Se asegura que la comunicación entre sistemas sea eficiente, ágil y que los datos puedan ser transmitidos correctamente. Sin embargo implementar la interoperabilidad puede ser complejo, especialmente cuando se trata de sistemas heredados o con diferentes estándares.
## Cumplimiento:
Se buscara disminuir el acoplamiento de los distintos componentes para implementar coorectamente una interoperabilidad en el sistema.
